## Hakkında

Koru MsSql Yedek, Microsoft SQL Server veritabanlarını ve dosyaları otomatik olarak yedekleyen, birden fazla bulut ve yerel hedefe yükleyen bir Windows masaüstü uygulamasıdır.

### Özellikler

- 🗄️ SQL Server veritabanı yedekleme (Full Backup)
- 📁 Dosya/klasör yedekleme (VSS Shadow Copy destekli)
- ☁️ Bulut depolama: Google Drive, OneDrive, FTP/SFTP
- 📦 7-Zip sıkıştırma (AES-256 şifreleme)
- ⏰ Zamanlanmış otomatik yedekleme
- 📧 E-posta bildirimleri
- 🔄 Otomatik güncelleme
- 🗑️ GFS Retention politikası (Grandfather-Father-Son)
